# Amazon-Sales-Sample-PowerBI-Dashboard
Interactive Power BI dashboard analyzing Amazon sales sample data.

This project is an interactive Power BI dashboard built using a sample Amazon sales dataset.  
The objective of this project is to analyze sales performance, profitability, returns, and regional trends using business-focused visualizations.

> Note: This project uses a publicly available sample dataset and is created for learning and portfolio purposes.
